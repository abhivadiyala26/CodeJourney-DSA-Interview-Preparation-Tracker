# 🚀 CodeJourney – DSA & Interview Preparation Tracker

CodeJourney is a full-stack web application that helps students and aspiring software engineers track their Data Structures & Algorithms (DSA) preparation, monitor coding consistency, and stay organized throughout their interview preparation journey.

The platform provides a centralized dashboard to manage coding progress, track topic-wise completion, maintain daily streaks, set preparation goals, and visualize overall performance through insightful analytics.

---

## ✨ Features

- 📊 Track solved DSA problems
- 🎯 Set learning goals and milestones
- 🔥 Maintain daily coding streaks
- 📚 Monitor topic-wise progress
- 📝 Save notes and revision materials
- 📈 Interactive analytics dashboard
- 👤 Secure authentication and user profiles
- 📱 Fully responsive design
- 🌙 Clean and modern user interface

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/codejourney.git
cd codejourney
```

### Install Dependencies

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
